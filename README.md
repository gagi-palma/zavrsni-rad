## Instalacija

1. Clone the repo
```sh
git clone https://github.com/gagi-palma/zavrsni-rad.git
```
2. Install NPM packages
```sh
npm install or npm i
```
3. Start JSON server
```sh
npm run server
```
4. Start project
```sh
run index.html
```
